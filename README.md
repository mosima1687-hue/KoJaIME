# KoJa IME

한국어식 발음으로 일본어를 입력하는 Windows용 입력기입니다.

예를 들어 한국어식으로 `리`, `쟈`, `갓코우니 이키마스`처럼 입력하면 일본어 kana와 한자 후보를 사용할 수 있습니다.

```text
리 -> り
쟈 -> じゃ
갓코우니 이키마스 -> 学校に行きます
```

## 다운로드

최신 설치파일은 GitHub Releases에서 받을 수 있습니다.

`KoJaIME-Setup.exe`를 다운로드해서 실행하면 됩니다.

## 주의

현재는 초기 테스트 배포판입니다.

코드 서명 인증서가 아직 없기 때문에 Windows SmartScreen에서 `알 수 없는 게시자` 경고가 뜰 수 있습니다.

공식 GitHub Release에서 받은 파일인지 확인한 뒤 `추가 정보 > 실행`을 선택해 설치할 수 있습니다.

## 지원 환경

- Windows 64-bit
- 일본어 언어/입력기 설치 필요
